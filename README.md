# Mini ISO 27001 Compliance & Risk Program

## Overview
This project simulates building a baseline ISO 27001:2022-aligned compliance and risk management program for a fictional SaaS startup, Sofunny101Cloud AI Solutions.

The objective is to assess security posture, identify risks, evaluate control implementation, and prepare audit-ready documentation.

## Scope
- Cloud infrastructure (AWS)
- SaaS tools (Google Workspace, GitHub, Slack)
- Employee endpoints
- Identity and access management
- Logging, monitoring, and incident response

## Frameworks Used
- ISO/IEC 27001:2022 (Primary)
- NIST 800-53 (Reference mapping)

## Key Deliverables
- Asset Inventory
- Risk Register
- ISO 27001 Control Matrix
- Security Policies
- Executive Compliance Summary

## Disclaimer
All content is fictional and sanitized for educational and portfolio purposes.
